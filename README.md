# ИИ-ассистент для компании по чистке и обслуживанию колодцев

Ссылка на проект: @ChistkaKolodzevBot

1. Описание проекта
Проект представляет собой интеллектуального ИИ-ассистента, разработанного специально для компании, специализирующейся на чистке и техническом обслуживании колодцев. Цель ассистента — автоматизировать первичное взаимодействие с клиентами, предоставлять точную информацию о услугах и записывать клиентов на выезд специалиста, тем самым повышая конверсию и снижая нагрузку на сотрудников.

Ассистент интегрирован в каналы коммуникации компании (сайт, мессенджеры) и способен вести осмысленный диалог на естественном языке, используя актуальную базу знаний.

2. Ключевой функционал
Осмысленное ведение диалога
ИИ-ассистент использует современные языковые модели (LLM) для понимания намерений пользователя и генерации релевантных, контекстуально точных ответов.
Ответы на основе базы знаний
Все ответы формируются на основе заранее подготовленной базы знаний, содержащей информацию об услугах, ценах, сроках выполнения работ, технических требованиях и часто задаваемых вопросах.
Запись клиента на встречу
По завершении диалога ассистент может предложить записать клиента на удобное время для выезда специалиста. Информация о встрече автоматически добавляется в Google Календарь компании.

3. Используемые технологии и сервисы
Компонент Описание: 
Qwen - Использовался для разработки системного промпта и структурирования базы знаний. Qwen помог адаптировать тон общения под специфику целевой аудитории и техническую направленность бизнеса.
Савви (Suvvy.ai) - Платформа для создания ИИ-ботов с поддержкой сложных сценариев диалога и высокой точностью ответов (от 95%). Савви обеспечивает 100% обработку обращений, интеграцию с внешними системами (в т.ч. Google Календарь), а также подключение к CRM и мессенджерам.

4. Преимущества решения
Высокая точность ответов — до 95% благодаря использованию надёжных LLM и качественной подготовке базы знаний.
Непрерывная доступность — ассистент работает 24/7 и отвечает на 100% обращений.
Автоматизация рутинных задач — запись на выезд, ответы на типовые вопросы, фильтрация «горячих» лидов.
Гибкая интеграция — легко подключается к сайту, Telegram, WhatsApp, CRM и другим корпоративным инструментам.
Возможности масштабирования
Расширение базы знаний для поддержки новых услуг (например, анализ воды, ремонт насосов).
Подключение к внутренней CRM для автоматического обновления статусов заявок.
Мультиязычная поддержка для работы в разных регионах.
